More ways to search ATI
© 2010

You can now install the ATI search engine directly into certain web browsers.

Firefox

You can perform contextual searches of ATI or type your search directly into the Firefox search window. Here's how to install it:

  1. Launch Firefox.
  2. If you had previously installed this plugin before Summer 2010, you must remove the old version. Here's how:
    1. Click on the button to the left of Firefox's search window. A pop-up list of search engines will appear.
    2. Select "Manage Search Engines..."
    3. Select "Access to Insight"
    4. Click "Remove"
    5. Click "OK"
  3. Visit any page at Access to Insight.
  4. Click on the button to the left of Firefox's search window. A pop-up list of search engines will appear.
  5. Select "Add Access to Insight".

From now on, whenever you select a chunk of text in your browser window you can search ATI for that text by right-clicking (or CTRL-clicking) on it, and selecting "Search Access to Insight for ...".

Opera

You can perform contextual searches of ATI or type your search directly into the Opera search window. Here's how to install it:

To install ATI search to Opera:

  1. Launch Opera.
  2. Visit any page at Access to Insight.
  3. Right-click (or control-click) on the ATI search window.
  4. In the pop-up menu, select "Create Search..."
  5. Use the name "Access to Insight".
  6. Type "ATI" as keyword.
  7. Click "OK".

From now on, whenever you select a chunk of text in your browser window you can search ATI for that text by right-clicking (or CTRL-clicking) on it, then selecting "Search with..." from the pop-up menu, and then choosing "Access to Insight".

Safari

You can perform contextual searches of ATI or type your search directly into a pop-up window. Here's how to install it:

  1. Launch Safari.
  2. Install the CustomSearch extension.
  3. Safari > Preferences > Extensions > CustomSearch
  4. Add a search named "Access to Insight" and paste the following chunk of text into its "URL" field:

    http://www.accesstoinsight.org/search_results.html?cx=015061908441090246348%3Aal1bklhbjbi&cof=FORID%3A9%3BNB%3A1&ie=UTF-8&q=#TEXT#

  5. Close the Preferences window.
  6. Right-click (or CTRL-click) on the toolbar. Select "Customize Toolbar..."
  7. Drag the button corresponding to the ATI search onto the toolbar.

From now on, you can search ATI in either of two ways: (1) select a chunk of text in your browser window, then right-click (or CTRL-click) and choose "Search Access to Insight" from the pop-up menu; or (2) click the toolbar button that you added in the last step, above.
