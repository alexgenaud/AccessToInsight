Shell Scripts
© 2009

These are some of the shell scripts that I routinely use to manage the website. They are most definitely as-is. They are guaranteed only to take up space on your hard drive. Beyond that, who knows!

bulk.tcsh
Prepares the Offline Edition (formerly called the "bulk download").
static.tcsh
Uses wget to generate static html pages of the site. (Used by bulk.tcsh .)
makeLists.tcsh
Creates the file manifests.
dpr_update.tcsh
Generates the static Digital Pali Reader javascript file.
